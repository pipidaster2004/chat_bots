Данный бот реализует взаимодействие с различными API языковых моделей, выложенных на сайте HUGGINGFACE.
Поведение бота:
1) доступные команды
    а)/start - приветственное сообщение от бота пользователю с его именнем в формате "Привет, {имя пользователя}! Выберите модель.", после чего в inline клавиатурах будет доступен выбор языковых моделей.
    б)/history - возвращяет последние 5 сообщений пользователя (если их меньше возвращает все что есть).
2) сообщение от пользователя:
    2.1) первым делом срабатывает обработчик бд и сохраняет сообщение
    2.2) срабатывает обработчик сообщения и отправляет запрос в API языковой модели ранее выбранной пользователем.
    2.3.1) бот обрабатывает ответ и переводит его в необходимую кодировку, после чего ответ модели отправляется пользователю.
    2.3.2) у ответа будут две inline кнопки: повторный запрос и ссылка на модель для ознакомления

Описание базы данных:
1) используется SQlite
2) бот взаимодействует только с таблицей messages, которая имеет следующие поля:
    2.1) id - уникальный номер сообщения
    2.2) user_id - идентификатор пользователя, который отправли сообщение
    2.3) message - сообщение пользователя
    2.4) dt - время отправки сообщения
3) для правильного взаимодействия с базой данных необходимо реализовать следующие функции:
    3.1) add_message - добавляет последнее сообщение пользователя в бд
    3.2) get_last_messages - возвращает последние 5 сообщений пользователя или все что есть, если их меньше
    3.3) setup_database - инициация подключения к базе данных
    3.4) close_database - закрытие подключения к базе данных

Бот работает асинхронно, поэтому необходимый минимум библиотек:
1) asyncio
2) aiohttp
3) AsyncSQLite или аналоги
